============================================================
REGLES ABSOLUES
============================================================

- LOGO IMMUTABLE : jamais toucher aux 5 PNG, jamais rendre de logo dans React
- PALETTE : jamais de hex en dur dans components. Toujours var(--inari-*) ou rgba(var(--inari-red-rgb), X)
- SURGICAL : pas de refactor global d'un code qui marche. Une correction = une modif minimale ciblee
- pnpm EXCLUSIF : jamais npm/yarn/bun. Aucun nouveau package sans justification + audit
- VERSIONS EXACTES : pas de ^ ni ~ dans package.json
- pnpm audit --prod OBLIGATOIRE apres chaque pnpm add
- CSS via tokens + Tailwind v4 @theme. Pas de !important sauf override WP/Kadence documente
- Security-first : CSP compatible, pas d'inline handlers, pas de secrets
- Pas de localStorage / sessionStorage / browser storage dans l'island
- Francais partout (code comments, UI text, commits)
